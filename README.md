# Let Me Go - 健康提醒助手

一个简单易用的Windows健康提醒应用，定时提醒你站起来走走，保护身体健康。

## 功能特点

- ⏰ **定时提醒**：自定义提醒时间段和间隔，定时弹出提醒窗口
- 🚫 **屏蔽时间段**：设置免打扰时间段（如午休时间）
- 🔔 **系统托盘**：后台静默运行，显示倒计时图标
- 🚀 **开机自启**：支持设置开机自动启动
- ⚙️ **配置灵活**：所有时间参数均可自定义
- 📦 **独立运行**：打包成exe后无需Python环境即可运行

## 安装使用

### 方式一：使用打包好的exe（推荐）

1. 下载 `let_me_go.exe`
2. 双击运行，首次启动会显示配置窗口
3. 设置你的提醒时间后点击"保存并启动"
4. 程序将最小化到系统托盘运行

### 方式二：从源码运行

1. 确保已安装 Python 3.7+

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行程序：
```bash
python let_me_go.py
```

### 方式三：自行打包

1. 安装依赖：
```bash
pip install -r requirements.txt
pip install pyinstaller
```

2. 打包成exe：
```bash
pyinstaller let_me_go.spec
```

3. 打包完成后，exe文件在 `dist` 目录下

## 配置说明

程序首次运行会显示配置窗口，你可以设置以下参数：

- **开始提醒时间**：开始提醒的时间（格式：HH:MM，如 09:00）
- **结束提醒时间**：停止提醒的时间（格式：HH:MM，如 18:00）
- **屏蔽时间段开始**：免打扰时间段开始时间（格式：HH:MM，如 12:00）
- **屏蔽时间段结束**：免打扰时间段结束时间（格式：HH:MM，如 13:30）
- **提醒间隔**：两次提醒之间的间隔时间（单位：分钟）
- **开机自动启动**：勾选后程序会在Windows启动时自动运行

### 配置文件

所有配置保存在程序目录下的 `config.json` 文件中，格式如下：

```json
{
  "start_time": "09:00",
  "end_time": "18:00",
  "block_start": "12:00",
  "block_end": "13:30",
  "interval_minutes": 60,
  "auto_start": false
}
```

你可以直接编辑此文件来修改配置，修改后需要重启程序生效。

## 使用技巧

1. **系统托盘操作**：
   - 右键托盘图标可打开设置菜单
   - 可以手动触发提醒
   - 可以退出程序

2. **重新配置**：
   - 右键托盘图标选择"设置"
   - 或使用命令行参数：`let_me_go.exe --config`

3. **开机自启**：
   - 在设置窗口中勾选"开机自动启动"
   - 程序会自动添加到Windows启动项

## 注意事项

1. 程序需要管理员权限来设置开机自启动（可选）
2. 确保防火墙允许程序运行
3. 配置文件 `config.json` 需要和exe在同一目录

## 常见问题

**Q: 程序没有提醒？**
A: 检查是否在设置的提醒时间段内，以及是否在屏蔽时间段内。

**Q: 如何关闭开机自启动？**
A: 右键托盘图标选择"设置"，取消勾选"开机自动启动"。

**Q: 程序无法启动？**
A: 检查是否缺少必要的依赖库，尝试重新安装 requirements.txt 中的依赖。
